本项目用于测试和分析微波噪声雷达的测距与测速算法。

## 目标
*   收集并整理相关的信号处理算法（如互相关、Keystone变换、DOA估计等）。
*   使用MATLAB/Python实现核心算法并进行仿真。
*   分析和比较不同算法在噪声雷达应用中的性能。
